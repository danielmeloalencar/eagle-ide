## 🚀 Instalando EAGLE IDE

Para instalar o Eagle IDE, siga estas etapas:

Windows / Linux:

```
$ npm install
```

## ☕ Rodando o Eagle IDE

Para rodar o Eagle IDE, siga estas etapas:

```
npm run electron:serve
```

Para compilar o Eagle IDE, siga estas etapas:

```
npm run electron:build
```
